2. Колірна схема

Тип колірної схеми вебсайту – монохромний. Ця схема використовує червоний колір, щоб створити відчуття комфорту та гостинності.

Обґрунтування загального вибору кольорів

Монохроматична колірна схема приємна для ока та добре підходить для мінімалістичного та стильного дизайну.

Роль окремих кольорів

#942404 (темно-червоний)

Цей колір використовується як акцентний колір. Він яскравий і привертає увагу, що робить його хорошим вибором для заголовків, кнопок та інших елементів, які треба виділити.

#F5F5F5 (білий дим)

Світло-сірий колір є нейтральним кольором, який не відволікає від контенту.

#343434 (темно-сірий)

Цей колір добре контрастує з іншими світлими кольорами сторінки.

3. Схема ліній, меж і фону

Фон сторінки світло-сірого кольору забезпечує контрастне відображення різних елементів інтерфейсу та робить сайт більш приємним для очей. Прямокутники білого кольору забезпечують фон для відокремлення різних частин інтерфейсу, таких як шапка сайту, посилання на гравців, форма для входу або регістрації та ін. Горизонтальні лінії сірого кольору на одному рівні з заголовками відокремлюють різні розділи головної сторінки. Прямокутники темно- червоного кольору позначають кнопки. Для групування елементів інтерфейсу використовуються принципи близькості та подібності.

4. Схема шрифтів

Шрифт Roboto — це шрифт без засічок. Він використовується на сайті для всіх основних елементів тексту, включаючи заголовки, абзаци та посилання. Шрифт має чіткі форми та прості пропорції, що робить його легким для читання. Шрифт Roboto також має широкий діапазон начертань, що дозволяє використовувати його для різних цілей.

5. Припущення та обмеження

Був використаний монохромний тип колірної схеми, що добре підходить для мінімалістичного дизайну, однак така кольорова схема може здатись трохи одноманітною. Також, можна розширити перелік шрифтів для більш різноманітного інтерфейсу.
